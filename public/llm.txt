# Rainz Weather - LLM Context

## About Rainz Weather
Rainz Weather is a hyperlocal weather application that provides accurate forecasts using multi-model ensemble forecasting from 7+ weather sources. The app combines data from ECMWF, GFS, Met.no, and other meteorological models to deliver reliable weather predictions.

## Key Features

### Free Features (All Users)
- Multi-model ensemble weather forecasts
- Real-time weather updates with minute-by-minute precipitation data
- Weather alerts and winter condition warnings
- Community weather predictions and leaderboard
- Prediction battles with other users
- Save multiple locations
- Pollen/Snow index (seasonal)
- Air Quality Index (AQI)
- Barometer readings
- Rain radar maps
- 10-day forecast
- Hourly forecast
- Detailed weather metrics

### Rainz+ Premium Features
- AI-enhanced weather data processing
- Push notifications with personalized weather updates
- Ad-free experience
- AI Weather Companion chat assistant
- Advanced display customization options
- Extended 14-day forecasts
- Premium leaderboard badge

## Premium Display Settings (Rainz+)
Subscribers can customize their weather display with these toggle options:
- Animated weather backgrounds
- Compact mode for condensed cards
- Show/hide "feels like" temperature
- Show/hide wind chill & heat index
- Show/hide humidity display
- Show/hide UV index
- Show/hide precipitation chance
- Show/hide dew point
- Show/hide pressure
- Show/hide visibility
- Show/hide sun times (sunrise/sunset)
- Show/hide moon phase

## Technology Stack
- Frontend: React, TypeScript, Tailwind CSS, Vite
- Backend: Supabase (PostgreSQL, Edge Functions, Authentication)
- Weather APIs: WeatherAPI.com, Tomorrow.io, Open-Meteo, Met.no
- Payments: Stripe for Rainz+ subscriptions
- PWA: Progressive Web App with offline support

## User Data & Privacy
- Location data is used only for weather forecasts
- Predictions and reports are stored for leaderboard rankings
- Rainz+ subscribers are not tracked by advertising networks
- All data is encrypted in transit and at rest
- GDPR and CCPA compliant

## Weather Predictions Game
Users can make daily predictions about tomorrow's weather including:
- High temperature
- Low temperature
- Weather condition

Points are awarded based on accuracy:
- +300 points if all 3 predictions correct
- +200 points if 2 predictions correct
- +100 points if 1 prediction correct
- +25 points for maintaining a prediction streak
- -100 points if all predictions wrong

## Prediction Battles
Users can challenge others to prediction battles:
- Battles expire at midnight on the day they're created
- Both users make predictions for the same location
- Winner receives bonus points
- Battles appear on the leaderboard
